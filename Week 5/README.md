# Week 5

This folder contains the Week 5 Flutter workshop project for a travel expenses app.

## What It Includes

- A `Expenses` stateful widget launched from `lib/main.dart`
- An `Expense` model with a generated UUID
- A `Category` enum for expense categories
- A hardcoded list of expenses
- An `ExpensesList` widget using `ListView.builder`
- An `ExpensesItem` widget that displays each expense in a `Card`

## Main Files

- `lib/main.dart`: app entry point
- `lib/expenses.dart`: main expenses screen and hardcoded data
- `lib/models/expense_model.dart`: expense data model
- `lib/widgets/expenses_list.dart`: list widget for expenses
- `lib/widgets/expenses_item.dart`: card widget for a single expense
