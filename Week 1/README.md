# Week 1

This folder contains the Week 1 Flutter workshop project. It demonstrates the first set of basic Flutter screens from the workshop instructions.

## What It Includes

- `Screen1`: a screen without a `Scaffold`
- `Screen2`: a screen using `Scaffold`
- `Screen3`: a `Scaffold` with a centered widget
- `Screen4`: a `Container` with a teal background
- `Screen5`: styled text using the Bebas Neue font at size 80
- `Screen6`: the final screen with an `AppBar`

The app currently launches `Screen6` from `lib/main.dart`.

## Main Files

- `lib/main.dart`: all Week 1 screen implementations
- `assets/fonts/BebasNeue-Regular.ttf`: font used by the styled text screen
- `pubspec.yaml`: Flutter project configuration and font declaration
